# 使用マニュアル

顔ジェスチャーコミュニケーションシステムの使い方です。

## 基本操作

### ジェスチャー一覧

| 動作 | 意味 | 使い方 |
|------|------|--------|
| 2回連続まばたき | はい / 決定 | すばやく2回まばたきする |
| 目を1秒以上閉じる | いいえ / キャンセル | 目を閉じて1秒待つ |
| 眉を上げる | メニュー表示 | 眉を上にあげる |
| 口を開ける | 選択を確定 | 口を大きく開ける |
| 頭を左に傾ける | 前の項目 / 戻る | 頭を左側に傾ける |
| 頭を右に傾ける | 次の項目 / 進む | 頭を右側に傾ける |

### 操作の流れ

```
1. 眉を上げる → メニューが表示される
2. 頭を傾ける → 項目を選ぶ
3. 口を開ける → 決定
4. 2回まばたき → はい / 目を閉じる → いいえ
```

## モード説明

### 待機モード
- 画面に「待機中」と表示
- 眉を上げるとメインメニューへ

### メインメニュー
1. **はい/いいえモード** - 質問に答える
2. **定型文モード** - よく使う言葉を選ぶ
3. **Audibleモード** - 本を聴く

### 定型文モード

カテゴリから選択:

- **基本**: 水、トイレ、看護師、先生
- **体調**: 痛い、頭痛、腹痛、気持ち悪い、めまい
- **環境**: 暑い、寒い、明るい、暗い
- **食事**: お腹空いた、お腹いっぱい、お茶
- **その他**: ありがとう、すみません、待って

### Audibleモード

- **口を開ける** → 再生/停止
- **頭を左に傾ける** → 30秒戻る
- **頭を右に傾ける** → 30秒進む
- **眉を上げる** → メニューに戻る

## キャリブレーション

個人差に合わせて感度を調整できます。

### 調整が必要な場合
- まばたきが認識されにくい → 目の閾値を上げる
- 口を開けても反応しない → 口の閾値を下げる
- 頭を傾けても反応しない → 傾き角度を小さくする

設定ファイル: `config/thresholds.yaml`

## 注意事項

- カメラに顔が映るようにしてください
- 照明が暗すぎると認識しにくくなります
- ゆっくり、はっきりと動作してください
